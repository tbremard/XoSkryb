TaskName   : Microphone deduplication and device validation
-------------------------------------------------------------------------
Last Update: 2026-03-19 19:35:00
-------------------------------------------------------------------------
Description: Windows PortAudio exposes the same physical microphone through multiple host APIs (MME,
  DirectSound, WASAPI), producing duplicate entries with different indices. Higher-index duplicates
  (e.g. index 26 for Samsung Q9U) crash on open. Fixed by deduplicating by name and validating each
  device by actually opening a test stream before accepting the selection.
